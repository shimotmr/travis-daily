# OpenClaw 每日動態 — 2026-02-13

本週 OpenClaw 安全議題集中爆發，Fortune、Bitsight、TechXplore 同步報導。NanoClaw 作為更安全的輕量替代方案兩週內衝上 7,000 GitHub stars。

## 🔥 五大關注項目

### 1. Fortune 深度報導：安全專家對 OpenClaw 感到不安
Fortune 引述多位資安專家，指出 OpenClaw「唯一的規則就是沒有規則」。Prompt injection、permission misconfiguration 是核心風險。

### 2. NanoClaw 爆紅
1/31 開源、MIT License，兩週 7,000+ stars。定位為解決 OpenClaw 最大安全問題的輕量版本，VentureBeat 大篇幅報導。

### 3. 30,000+ 個 OpenClaw 實例暴露公網
Bitsight 研究發現 1/27-2/8 期間超過三萬個未保護的 OpenClaw 部署直接暴露在網際網路上。

### 4. AI Agent 安全基礎設施需求爆發
Cloudflare 因 AI Agent 安全需求股價漲 5%。AI Agent 時代的安全問題正在成為投資主題。

### 5. Seedance 2.0 尚未開源
ByteDance 最新 AI 影片模型支援 20 秒生成、多模態輸入，但**目前為閉源**，僅透過 Dreamina/CapCut 使用。開源替代仍推薦 WAN 2.2。
